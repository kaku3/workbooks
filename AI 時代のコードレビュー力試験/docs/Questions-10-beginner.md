# 初級問題（Beginner）

`sample-ec` プロジェクトを clone して、コードを読んだ上で回答してください。

> **AI ツールの利用は自由です。** ただしすべての設問で「なぜそう判断したか」を自分の言葉で書いてください。

## B-01: `utils.js` が育ちすぎた

**依頼内容**

「メール送信処理を追加したい。共通処理は `utils.js` に置くルールだから、そこに書いておいて」

**確認してほしいファイル**

- `utils.js`

**設問**

1. `utils.js` にはどのような処理が含まれていますか。担っている責務をすべて列挙してください
2. この状態でメール送信処理を `utils.js` に追加することに問題はありますか
3. `utils.js` をどのように分割しますか（ファイル名と、そこに入れる関数の概要を示してください）
4. なぜ「`utils.js` が何でも屋になる」という現象が起きると思いますか。技術的な理由と、チームの習慣・文化の両面から考えてください

## B-02: `utils.js` と `common.js`、どっちが正しい？

**依頼内容**

「金額表示を変えたい。`¥` の前後にスペースを入れてほしい」

**確認してほしいファイル**

- `utils.js`
- `public/common.js`

**設問**

1. 金額フォーマットに関係する関数は、このプロジェクトにいくつありますか。それぞれどのファイルにありますか
2. 依頼された変更をするとしたら、どのファイルを修正しますか。その判断理由を述べてください
3. 「修正したのに一部の画面だけ反映されない」という不具合が起きた場合、原因として何が考えられますか
4. この状態（同じ関数が複数ファイルに存在する）になった原因として、何が考えられますか

## B-03: 消費税、実はいくらですか？

**依頼内容**

「カート画面・チェックアウト画面・注文確定後の金額が微妙にズレていると言われた。調べてほしい」

**確認してほしいファイル**

- `public/cart.html`（`// 消費税` のあたり）
- `public/checkout.html`（`// 消費税を再計算` のあたり）
- `app.js`（`// 消費税（切り捨て）` のあたり）

**設問**

1. 税計算のコードは3箇所あります。それぞれどのような計算をしていますか。違いをすべて列挙してください
2. 商品単価 `1,980円` の商品を1個購入した場合、税額はそれぞれいくらになりますか。実際に計算してください
3. 「消費税率を 10% から 8% に変更する」という仕様変更が来た場合、このコードではどこを修正する必要がありますか
4. なぜこのような「同じルールの実装がバラバラ」という状態になったと思いますか
