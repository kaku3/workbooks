## 採点プロンプト

> 以下の指示に従って、受験者の回答を採点してください。

受験者の回答ファイルにある S-01〜S-04 の全設問を、模範解答・採点基準に従って採点してください。

**採点方針**

- 模範解答と一言一句一致している必要はない。「体験に基づいた自分の言葉」で書かれているかを重視する
- 「なぜそう判断したか」が書かれているかを必ず確認する
- AI の出力をそのまま貼り付けただけで「なぜそう判断したか」がない場合は評価を下げる
- 「気づかなかった」「うまくできなかった」という正直な回答も、気づきとして評価する

**出力形式**

設問ごとに以下の形式で出力してください。

```
## {設問番号}：{設問タイトル}

**評価**：◎ / ○ / △ / × のいずれか

**コメント**：
（採点理由・良かった点・改善できる点を 3〜5 文で）
```

全設問の採点が終わったら、以下のサマリーを出力してください。

```
## サマリー

**総評**：（全体を通じた印象を 3〜5 文で）

**強み**：（この受験者が特によくできていた観点）

**次のステップ**：（この受験者が次に取り組むとよいこと）
```
