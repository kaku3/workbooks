# 中級問題（Intermediate）

`sample-ec` プロジェクトを clone して、コードを読んだ上で回答してください。

> **AI ツールの利用は自由です。** ただしすべての設問で「なぜそう判断したか」を自分の言葉で書いてください。

---

## M-01: プレミアム割引、なぜか効かないユーザーがいる

**依頼内容**

「佐藤花子さんからクレームが来た。プレミアム会員なのにカートで割引が表示されない」

**設問**

1. 「佐藤花子さんの割引が効かない」原因を調査してください。コードを根拠に説明してください
2. 修正するとしたら、どこをどう直しますか。影響が最小のアプローチを選んでください
3. このバグはなぜ生まれたと思いますか。また、同様の不整合が他に潜んでいる可能性はありますか

---

## M-02: トークンの実装が2種類ある

**依頼内容**

「ログインしても一部の API で401が返ってくることがある。トークン周りを確認してほしい」

**設問**

1. 「一部の API で 401 が返る」原因を調査してください
2. この状況をどう整理しますか（どちらを残すか、どのように統一するか）
3. このような「実装が 2 系統ある」状態はなぜ生まれると思いますか。また今後どう防ぎますか

---

## M-03: 在庫が減らない注文処理

**依頼内容**

「注文は通るのに在庫が減っていないと言われた。確認してほしい」

**設問**

1. 「注文は通るのに在庫が減らない」原因を調べてください
2. 修正するとしたらどのようなアプローチを取りますか。修正前に確認・検討すべきことも述べてください
3. この問題を設計の観点から振り返ると、何が根本原因だと思いますか
