# アクションフェーズ 詳細仕様

## 概要

全プレイヤーのコマンドを確定後、同時に実行するフェーズ。  
実行結果はアニメーションで順次表示される。アニメ完了後に次ターンのコマンドフェーズへ移行する。

---

## 処理実行順序

アクションフェーズの各処理は以下の順序で実行される。

```
0) 防御バフ (チャフ / 装甲板)
1) 収縮ダメージ   — 現在無効
2) 移動
3) 機雷チェック
4) 補給
5) ソナー
6) 武器 (魚雷 / 追尾魚雷 / 散弾)
7) 設置 (デコイ / 機雷)
8) ドッグファイト判定
9) 前方警戒判定
10) デコイ寿命
11) 脱落チェック
```

---

## 各ステップ詳細

### 0) 防御バフ

| 操作 | 効果 |
|---|---|
| チャフ | `chaffActive = true`。同ターンの追尾魚雷 1 発を無効化、安除がみにフラグリセット。在庫消費はこのタイミング。 |
| 装甲板 | `armorActive = true`。同ターンの散弾 1 発を無効化、安除がみにフラグリセット。在庫消費はこのタイミング。 |

### 1) 収縮ダメージ

**現在無効**。フィールド収縮機能はコメントアウト中。

### 2) 移動

全プレイヤーの移動コマンドを同時実行。  
cat = `move` の操作をキュー順に適用する。

| 操作 | 処理 |
|---|---|
| 前進 | 向き方向に 1 マス、10 以外はクランプ |
| ←旋回 / 旋回→ | 向きを 90° 変更 |
| ←横移 / 横移→ | 左または右方向に 1 マス移動 |

### 3) 機雷チェック

全プレイヤーの移動後位置を展開中の機雷リストと照合わせる。

- 同一セルに装備側の機雷があり、躢踏者が設置者以外の場合→ **1 ダメージ** で機雷消滅。
- 爆発イベント (`explosion`) は全員公開。

### 4) 補給

補給ポイントと同一セルにいるプレイヤーに効果を適用。

| 種別 | 効果 |
|---|---|
| ammo | 全在庫初期値に回復 |
| repair | HP +1 (最大値以下) |

### 5) ソナー

指定セルを中心に Chebyshev 距離 1 (3×3) をスキャン。

- デコイを持つ敵は探知されない。
- ヒット情報は発射者のみ受信する（`public: false`）。
- アニメーション中はヒットマーカー (× + 「機影」ラベル) を展示。

### 6) 武器

#### 魚雷 (torpedo)

- 発射時の向き方向に直線でグリッド内最大 10 マス飛赲。
- 最初にヒットした敵に **2 ダメージ**、そこで止まる。
- 発射者のみプライベートイベント (`torpedo_fire`) と射線方向の公開イベント (`attack_leak`) をプッシュ。
- アニメ：発射者画面に確射線と矢印を表示。

#### 追尾魚雷 (guided)

- 指定座標にいる敵に **1 ダメージ**。
- 指定座標は発射者の向きの **45° 扇形内** のみ有効。扇形外はサーバー側で無視。
- チャフバフありの敵に命中した場合：チャフが消費されダメージなし。
- 発射者のみプライベートイベント (`guided_fire`)。
- アニメ： rAF アニメーションで魚雷が発射地 → 標的地に射動、2.5 秒。トレイル（点線）＋グロードットで表現。

#### 散弾 (shotgun)

- 向きの前方、左斜め前、右斜め前の 3 セルに **1 ダメージ**。
- 装甲板バフありの敵に命中した場合：装甲板が消費されダメージなし。
- 全員公開イベント (`attack_leak` op=shotgun)。
- アニメ：発射方向の扇形 3 セルをオレンジで色塗り＋林畜。発射元から各セルへの矢印のアニメ。

### 7) 設置

| 操作 | 処理 |
|---|---|
| デコイ | 指定座標にデコイ追加。turnsLeft = 2。発射者のみプライベートイベント |
| 機雷 | 指定座標に機雷追加。発射者のみプライベートイベント |

在庫消費はステップ 7 の後に一括指定される（chaff / armor はステップ 0 で消費済み）。

### 8) ドッグファイト判定

- 既存ドッグファイトペアの距離が 4 以上になった場合→解除。
- Chebyshev 距離 **3 以内**の新たなペアを検出しドッグファイト開始。
- どちらか一方が既にドッグファイト中の場合は新規ペアはスキップ。
- イベントは両者にプライベート送信（`dogfight_start`, `public: false`）。
- ターン 1 はドッグファイト判定をスキップ。

### 9) 前方警戒判定

自機の向きの前方 1～3 マスに敵がいる場合、自機だけに警告フラグを設定。

| 距離 | 警告レベル | 盤面表示 |
|:---:|---|---|
| 1 マス | `critical` | 赤橙 |
| 2 マス | `near` | 橙 |
| 3 マス | `far` | 淡黄 |

### 10) デコイ寿命

全デコイの `turnsLeft` を -1 し、0 になったものを削除。

### 11) 脱落チェック

`hp <= 0` のプレイヤーを脱落させ `alive = false, respawning = true` にする。

- 脱落イベントは全員公開。
- キルした側の `kills` をインクリメント。3 に達した時点で `winner` を記録。
- 脱落プレイヤーは次ターン開始時に外縁ランダム位置で HP 全回復・在庫全回復で復活。

## アニメーション実装

- アクションフェーズ開始後、1800ms のフェーズオーバーレイ表示後にアニメ起動。
- イベントキューを順次処理し、各イベントのおもにおも表示時間後に次へ進む。
- アニメ中は操作パネルを隠す。

### イベント表示時間

| イベント種別 | 表示時間 |
|---|---|
| eliminated | 4000 ms |
| damage | 3000 ms |
| torpedo_fire | 3000 ms |
| guided_fire | 2500 ms |
| attack_leak (guided) | 2000 ms |
| explosion | 2800 ms |
| sonar | 3000 ms |
| move (simple) | 1200 ms |
| move (その他) | 2500 ms |
| sound_leak | 400 ms |
| その他 | 1500 ms |

### イベント別アニメ詳細

| イベント | 公開範囲 | 描画内容 |
|---|---|---|
| move | 発動者のみ | 移動軌跡矢印 + 現在位置円弧 |
| sonar | 発動者のみ | スキャン範囲とヒットマーカー |
| damage | `source !== mine` の場合全員公開、機雷は死者のみ | 被弾セル赤塞り + `-N` テキスト |
| torpedo_fire | 発動者のみ | 確射線（オレンジ）+ 矢印 |
| guided_fire | 発動者のみ | rAF 移動アニメ (トレイル + グロードット) |
| attack_leak (shotgun) | 全員 | 扇形セルオレンジ塑り + 矢印 |
| explosion | 全員 | 爆発円弧オレンジ |
| eliminated | 全員 | 脱落位置に ✕ アイコン |
| respawn | 全員 | 複活位置に輪郭表示 |
| buff (chaff/armor) | 発動者のみ | アイコン + ラベル |

## アクションフェーズ中の可視情報

| 情報 | 表示対象 |
|---|---|
| 自機の位置・向き（アニメ中は巡回前の位置から逐次反映） | 常に表示 |
| 経路矢印 | 左優後に描画 |
| 武器效果エリア | 公開範囲のプレイヤーに表示 |
| ダメージ (`-N`) | `source !== mine` の場合全員公開 |
| 脱落アニメ | 全員 |

---

## ターンログ

各ターンのアクション結果はターンログに記録される。

記録内容例：

```
プレイヤー1 が 魚雷 で 2 ダメージ (HP:2)
プレイヤー1 がキル (腨1キル)
プレイヤー2 が弾薬を全回復
プレイヤー3 のチャフが追尾魚雷を無効化
```

---

## アクション完了後の処理

1. アニメ完了後、ホストが `advanceToNextTurn()` を呼びターンインクリメント。
2. `prepareNextTurn()` により復活待機中のプレイヤーが外縁に復活。
3. 全プレイヤーの `time`, `sonarResults`, `forwardWarning`, `commandQueue`, `commandConfirmed` をリセット。
4. `phase = 'command'` に戻りコマンドフェーズへ移行。
