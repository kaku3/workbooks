# 潜水艦ゲーム 操作パネル仕様 v5.0

## コンセプト

カードデッキ方式（v4.x）を廃止し、**パネル操作方式**へ移行。

| 旧（カード制） | 新（パネル制） |
|---|---|
| 毎ターン山札を選んで2枚ドロー | ドローなし。常に全操作が見えている |
| 手札次第で行動が変わる | 在庫量次第で戦略が変わる |
| 「何が引けるか」のランダム性 | 「何を選ぶか」の純粋な判断 |

### 設計方針

- **同時プロット維持**: 全員が同時に作戦を組み立て、一斉公開
- **単位時間 10**: 1ターン = 10単位時間。使い方で戦略が分かれる
- **向き管理の重要性**: 旋回コスト3により「今どっちを向いているか」が常に重要
- **在庫管理**: 武器・防御はパネルに残量表示。補充ポイントで回復

---

## 単位時間コスト一覧

### 移動（在庫なし・常時使用可）

| 操作 | 単位時間 | 備考 |
|---|---|---|
| 前進 | 1/マス | 何マスでも可。連続指定可 |
| 旋回（左/右） | 3 | 90°回転。コスト高。向き管理が戦略の核心 |
| 横移動（左/右） | 2/マス | 向きを変えずに移動。DF中の回避に有効 |

> **旋回コスト3の意図**:
> 「魚雷を撃つために向きを変える → 旋回3 + 魚雷1 = 4」かかる。
> 追尾魚雷・横移動・デコイなどの「向き不問」アクションに存在意義が生まれる。

### 索敵（在庫なし・常時使用可）

| 操作 | 単位時間 | 効果 |
|---|---|---|
| ソナー | 1 | 任意座標を指定。半径3マス以内に**敵・デコイ**がいれば反応。副作用なし |

> ソナーは1種類に統一（パッシブ・アクティブ区別廃止）。
> 任意座標指定なので「補給ポイント周辺を確認する」「前方の角を確認する」が可能。

### 武器（在庫制）

| 操作 | 単位時間 | 初期在庫 | 効果 |
|---|---|---|---|
| 魚雷 | 1 | 3 | **前方のみ**、射程6マス直線攻撃。HP-2。向き必須 |
| 追尾魚雷 | 2 | 2 | 任意座標指定。発射時点で指定位置に**敵またはデコイ**がいれば追尾。HP-1。チャフで無効化 |
| 散弾 | 1 | 2 | 前方扇状（3方向各3マス）。HP-1。チャフ無効。装甲板で無効化 |

**じゃんけん関係**:
```
魚雷（向き必須・高威力）
  ↑ 向き調整コストが大きい弱点
追尾魚雷（向き不問・中威力）← チャフで無効
散弾（向き不問・低威力・近距離）← 装甲板で無効
```

### 設置系（在庫制）

| 操作 | 単位時間 | 初期在庫 | 効果 |
|---|---|---|---|
| デコイ | 1 | 2 | 任意マスに囮配置。ソナーに反応・追尾魚雷を引き寄せる |
| 機雷 | 1 | 2 | 任意マスに設置。敵が半径1マス内に入ると爆発 → 半径2マス内全員HP-1 |

### 防御系（在庫制）

| 操作 | 単位時間 | 初期在庫 | 効果 |
|---|---|---|---|
| チャフ | 1 | 2 | このターン、追尾魚雷の追尾を無効化（デコイへの転換も防ぐ） |
| 装甲板展開 | 2 | 2 | 次のダメージを-1（散弾ダメージは0に。魚雷は2→1に） |

## プロット方法

1. **時間バー**を見ながら、使いたい操作をパネルから選ぶ
2. 移動・攻撃を**キューに積む**（順序が重要）
3. 10単位を使い切るか「確定」ボタンで送信
4. 全員確定 → 一斉解決（同時プロット）

### プロット例

```
作戦: 「前進3マス → ソナー(F5指定) → 追尾魚雷(G5) → 前進1マス」
       (1×3)    +  (1)           + (2)            + (1×1)    = 8単位
       ※残り2単位は温存（何もしない）
```

## 補充ルール

補充ポイントに**停止**すると、そのターン終了時に在庫が補充される。

| 補充ポイント | 補充内容 |
|---|---|
| 武器補充 | 魚雷+2、または追尾魚雷+1、または散弾+2（1択） |
| 修理 | HP+1 |
| 防御補充 | チャフ+2、または装甲板+1（1択） |
| 設置物補充 | デコイ+2、または機雷+2（1択） |

> 補充ポイントは全員に見えているので「補充に向かう動き」も情報になる。

## 作戦フレームワーク

「今ターン自分はどのパターンか」を意識することで判断が速くなる。

### パターンA: 敵の位置不明（索敵フェーズ）

**ゴール**: 敵を見つけ、補充ポイントを確保する

```
時間配分例:
  前進5 + ソナー1 + ソナー1 + 前進3 = 10
  → 移動しながら2箇所索敵
```

**有効なアクション**:
- ソナーを複数回使い、三角測量的に絞り込む
- デコイを置いて「自分がここにいるかも」と誤認させる
- 補充ポイント方向に移動しながら索敵

---

### パターンB: 敵の位置確定（攻撃フェーズ）

**ゴール**: 向き・距離を合わせて攻撃 → 即離脱

```
時間配分例（魚雷を使う場合）:
  旋回3 + 前進2 + 魚雷1 + 横移動2×1 + 前進2 = 10
  → 向き変えて→接近して→撃って→横にずれて→離脱

時間配分例（追尾魚雷を使う場合）:
  前進5 + 追尾魚雷2 + 前進3 = 10
  → 旋回不要なのでシンプル。ただしチャフで無効化リスク
```

**有効なアクション**:
- 魚雷: 旋回コストが重い。事前の向き管理が重要
- 追尾魚雷: 向き不問だがコスト2・チャフリスク
- 散弾: 距離3以内限定。DF中に最も使いやすい

---

### パターンC: ドッグファイト中

**ゴール**: 相手の攻撃を回避しながら先にダメージを与える

```
時間配分例:
  横移動2 + 散弾1 + チャフ1 + 前進6 = 10
  → 位置をずらして散弾・チャフ張りながら逃走準備
```

**有効なアクション**:
- **横移動**: 向きを変えずに位置をずらす → 魚雷の射線を外す
- **散弾**: 向き調整不要 → DF中の基本攻撃
- **チャフ**: 追尾魚雷への対抗。DF中は相手も撃ってくるので保険として有効
- **装甲板**: 散弾ダメージを無効化。「あえて食らう」選択肢も

---

## ドッグファイト仕様（v5.0）

| 項目 | 値 |
|---|---|
| 開始条件 | チェビシェフ距離3以内（ターン1を除く） |
| 情報公開 | お互いの座標・向きが全公開 |
| 解除条件 | 距離4以上になる（逃げるコストが重い点がポイント） |

**逃走コストの現実**:
- 反転（逃げ向きに旋回）= 3単位
- 横移動で距離を稼ぐ = 2単位/マス
- 緊急離脱（4マス移動）= コスト対比で効率的

旋回コスト3の重さにより、「真後ろに回り込まれたら逃げにくい」状況が自然に生まれる。

## 勝利条件（v5.0）

| モード | 条件 |
|---|---|
| スタンダード | 先に3キルで勝利 |
| タイムアタック | ターン10終了時の最多キル数 |

**復活ルール**:
- 撃沈されたターン: 観戦
- 次ターン開始: ランダム外縁マスに復活
- 復活時の在庫: 全在庫を初期値にリセット（ペナルティなし・ただし位置不利）

> **v5.0の変更点**: 燃料ペナルティを廃止。代わりに「位置不利（外縁スタート）」のみ。
> 在庫を全回復することで「復活直後は全力で戦える」快感が生まれる。

## v4.2 → v5.0 変更サマリ

| 項目 | v4.2（カード制） | v5.0（パネル制） |
|---|---|---|
| 手札 | 毎ターン2枚ドロー | なし |
| 行動リソース | 燃料（上限10） | 単位時間（上限10） |
| 旋回コスト | 2 | **3**（向き管理の重要性が増す） |
| 横移動 | カード必要 | 常時パネルから可 |
| 索敵 | 手札次第（種類あり） | **ソナー1種類・常時使用可** |
| 武器入手 | 手札次第 | **在庫制・パネル表示** |
| 補充 | デッキ循環 | 補充ポイントで在庫回復 |
| 復活在庫 | 燃料ペナルティあり | **全在庫リセット** |
