# シミュレーション 04 — 弾数無制限プレイの分析と適正弾数の導出

## 目的

初期弾数を「弾数無制限」と仮定してプレイをシミュレーションし、ゲーム1試合で消費される武器使用回数の分布を推定する。  
その結果をもとに「補給なしで約8〜10ターン戦える」「補給ポイント1〜2回訪問すればゲーム終盤まで戦える」適正弾数を決定する。

---

## ゲーム基本パラメータ

| 項目              | 値                   |
|-------------------|----------------------|
| グリッド          | 10×10 = 100マス      |
| プレイヤー数      | 3人想定              |
| 1ターン時間       | 10 単位時間          |
| HP                | 4                    |
| 勝利条件          | 3キル先取            |
| ターン上限(想定)  | 20〜30ターン         |
| 補給ポイント訪問  | 全ゲームで1〜2回程度 |

## ターン消費の内訳推定

### フェーズ別プレイスタイル

| ゲームフェーズ | ターン範囲 | 主な行動           |
|---------------|------------|--------------------|
| 初動期        | T1〜T6     | 移動・索敵中心      |
| 中盤交戦期    | T7〜T18    | 武器フル活用        |
| 終盤          | T19〜T25   | 決着・補給巡り      |

### 1ターンあたりの時間配分（平均）

```
前進系移動:  4 単位時間 (forward×4 想定)

```

| shotgun  | 1      | 0.5〜1.0             | 10〜20                 |

## ヒット率の分析

### torpedo（通常魚雷）

- **条件**: 前方45°コーン内を指定、直線飛翔
- **視点**: ソナーで敵位置概知 → 1マス誤差で外れる
- **ヒット率推定**: 15〜25%（ソナー情報あり）/ 5〜10%（盲撃ち）
- **必要ヒット数**: 全敵HP = 3人×4HP = 12 → 1発2ダメージなので6ヒット
- **必要発射数**: 6 ÷ 0.20 = **30発**（ソナー活用時）

### guided（追尾魚雷）

- **条件**: 指定座標±チェビシェフ2以内に敵がいたらヒット
- **ヒット率推定**: 35〜55%（ソナー後即撃ち）
- **必要ヒット数**: 12（1発1ダメージ）
- **必要発射数**: 12 ÷ 0.45 = **27発**

### shotgun（散弾）

- **条件**: 前方3セルが対象、近距離必須
- **ヒット率推定**: 50〜70%（ドッグファイト中）/ 10〜15%（遠距離）
- **必要発射数**: 12 ÷ 0.55 = **22発**（接近戦）

### decoy / mine / chaff / armor

- 防御・トリック系。1試合で**5〜10回**使えれば十分。

## 実質的な武器消費（補給ポイント考慮）

補給ポイントで在庫が全回復するため：

```
実際に必要な初期在庫 = (補給まで戦う期間) × (1ターンあたり使用数)
```

補給まで約8〜10ターン想定：

| 武器     | 必要回数(8T) | 必要回数(10T) | 推奨初期数 |
|----------|-------------|--------------|------------|
| torpedo  | 16〜24      | 20〜30       | **8**      |
| guided   | 8〜12       | 10〜15       | **4**      |
| shotgun  | 4〜8        | 5〜10        | **4**      |
| decoy    | 4〜6        | 5〜8         | **4**      |
| mine     | 2〜4        | 3〜5         | **3**      |
| chaff    | 4〜6        | 5〜8         | **4**      |
| armor    | 2〜4        | 3〜5         | **3**      |

## 3人ゲーム シミュレーション（代表例）

### プレイヤー特性

| ID | スタイル     | 主戦略                          |
|----|-------------|--------------------------------|
| P1 | ラッシャー   | 前進+torpedo連射+近接shotgun    |

```

T07-T12（中盤）: P1がtorpedo多用、P2がguidedで正確狙撃、P3がmine展開
```

## 分析まとめ

### torpedo が突出して多用される

- コスト1、前方45°と使いやすい → 毎ターン2〜3発が標準
- ヒット率は低いが「撃って当たったらラッキー」プレイが自然発生
- **→ 初期弾数は他の2〜3倍必要**

### guided はソナー情報との組み合わせがキー

- 指定座標±2以内をスキャンするため、ソナー直後に撃てば当たりやすい
- でも guided コスト2 + sonar コスト1 = 3単位 = 1ターンの約1/3
- **→ ソナーを引いた後の即撃ちを繰り返すプレイスタイルが最強**
- ユーザーが予測した「索敵が先 → 追尾魚雷を無理やり当てる」は正確

### mine/decoy は「戦場整形」に使える

- 設置が完了するとゲームが有利になるが、相手に踏ませるまで時間がかかる
- **→ 3〜4個で十分。多すぎるとゲームテンポが落ちる**

## 最終推奨弾数

```js
export const INITIAL_INVENTORY = {
  torpedo: 6,   // 補給まで8Tで12〜18発使う想定 → 余裕を持って6
  guided:  3,   // 補給まで8Tで6〜10発想定 → 3 + 補給
  shotgun: 3,
  decoy:   3,
  mine:    3,
  chaff:   3,
  armor:   3,
};
```

> **torpedo を 6 にした根拠**: 毎ターン 1〜1.5 発、8T で 8〜12 発が目安。  
> 補給ポイントまで「ギリギリ足りる」くらいにしたいが、  
> あまり少ないと初期から補給に追われてゲームにならないため 6 を下限とした。  
> 体感として多すぎると感じた場合は 4〜5 に減らすこと。

## ゲームデザイン上の考察

### 「索敵 → 追尾魚雷」が最強パターンになる問題

- ソナー(1u) → 追尾魚雷(2u) = 3u で確率45%ヒット × 反復
- これを3〜4ターン続けるだけでキルが取れる
- 対策: **チャフの使いどころを高める**（チャフ1個で追尾を1回無効化）
- または: guided の「指定座標固定」をもっと厳しくする（±1以内のみ追跡など）

### 機雷の評価が低い問題

- 設置コスト1だが、相手が踏む確率は低い
- 補給ポイント周辺に設置するのが唯一の強用途
- → **補給ポイント入口の機雷はゲームを面白くする**
