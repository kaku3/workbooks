# 潜水艦ゲーム

## コンセプト

3~6人で遊べる同時プロット制ゲーム。
github pages で公開できる static html ベース + PeerJS通信対戦。

推理＋予測重視

カードゲーム＋ボードゲームのハイブリッド
コマンドフェーズとアクションフェーズで進行

### コマンドフェーズ

カードドロー、「行動時間（コスト）」に収まる範囲で、制限時間内に各自行動を決める。
時間切れの場合は、そのターンの行動は行われない。

### アクションフェーズ

コマンドフェーズの内容に沿って、一斉にアクションを実行する。
ホスト側でアクション制御を行い、各クライアントに配信することで同期を取る。

## ゲームの概要

ボード上にランダムで各プレイヤーの潜水艦が配置される。
自分以外の潜水艦の場所は分からない。
ソナーカードなどで相手の場所を探る。

攻撃カードを使うと近くにいるプレイヤーには場所がバレる。
