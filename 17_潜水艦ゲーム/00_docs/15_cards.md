# カード仕様 v4.2

## 9. カード山札バリエーション

### 9-0. 固定コマンド（仮想カード）— 常時使用可能

手札を消費せず、**燃料のみ**で毎ターン使えるベースアクション。
デッキには含まれず、コマンドフェーズで画面上部に常時表示される。

| コマンド名 | 燃料 | 効果 |
|---|---|---|
| ← 左旋回 | **2** | 左に90°旋回（移動なし） |
| ↑ 前進   | **2** | 向いている方向に1マス移動 |
| → 右旋回 | **2** | 右に90°旋回（移動なし） |

> **設計意図**: 旋回=前進=2燃料で等価。「旋回しながら前進」が自然にできる。
> 旋回1回+前進4回=10燃料（ターン上限ぴったり）。1ターンに複数回使用可能。
> 高出力推進バフの影響も受ける。

### 9-1. v4.2 カード設計コンセプト

#### じゃんけん関係図

攻撃・防御・索敵が連動する「推理 → 選択 → 当てに行く」フロー。

```
【当てに行く3択】

  確定情報あり → 魚雷（高威力・1マスずれたら外れ）
  だいたいわかる → 誘導魚雷（追尾あり・チャフで無効化される）
  情報なし → 散弾（広域・低威力・自艦位置が全員バレ）

【対抗関係】

  誘導魚雷 ─────→ チャフ展開で無効化
  散弾（位置バレ） → 使った後は追撃されやすい（装甲や煙幕で対策）
  魚雷 ──────→ 事前に煙幕を張れば情報を取らせない

【索敵フロー】

  追跡弾で命中タグ → 2ターン間、相手の位置が毎ターン通知
  ↓ タグが付いた相手には魚雷が最強
  ↓ 相手は煙幕で1ターンタグ通知を遮断できる
```

> **ドッグファイト（DF）**: チェビシェフ距離 **3マス以内**で自動開始。
> お互いの位置・向きが開示される近接戦闘モード。魚雷の射程が-1。

### 9-2. デッキ構成（v4.2 標準、計35枚）

| カテゴリ | 枚数 | 山札 |
|---|---|---|
| 移動系 | 12枚 | 移動山札 |
| 索敵系 | 7枚 | 索敵山札 |
| 攻撃系 | 8枚 | 攻撃山札 |
| 特殊系 | 8枚 | 特殊山札 |

> 各ターン、プレイヤーは2つの山札カテゴリを選び、各1枚ドローする。

**v4.1→v4.2 削除・統合カード**:

| 削除カード | 理由 |
|---|---|
| 広域爆雷・連装魚雷 | → **散弾**に統合（広域攻撃を1種類に絞る） |
| 機雷敷設 | 設置待ちの戦術が複雑になるため削除 |
| 水中マイク | 情報量が少なく使いにくいため削除 |
| 追跡ブイ | → **追跡弾**に発展（場所固定→敵艦に直接タグ） |
| EMP | 複数ターン無効化は混乱を招くため削除 |
| 強化ソナー・魚雷追尾装置・連装砲準備 | バフ特化カードを絞り込み |

### 9-4. 索敵系カード（7枚）

| カード名 | ID | 燃料 | 枚数 | 効果 | 副作用 |
|---|---|---|---|---|---|
| パッシブソナー | passive_sonar | 2 | 3 | **任意の座標を指定**し、半径3マス以内を索敵（1ターン即時） | なし |
| アクティブソナー | active_sonar | 3 | 2 | 半径5マス以内を全索敵（1ターン即時） | 自艦の大まかな位置が全員に漏れる |
| 追跡弾 | tracking_shot | 2 | 2 | 前方3マスへ音響弾を発射。**命中した艦には2ターン間追跡タグ**（毎ターン開始時に位置を通知） | 外れても発射方向は漏れない |

> **パッシブソナーの使い方**: 自艦起点ではなく**任意座標を指定**して索敵するため、DF距離に入らずに遠方を調べられる。
> 「補給所付近(D4)に誰かいるか確認する」「壁沿いにいそうな座標を調べる」など情報戦の核心。
> コスト2（v4.1から+1）は座標指定の自由度を補正。
>
> **追跡弾の狙い**: 命中 → 2ターン間、相手の位置が毎ターン自分に通知される。
> その間に**魚雷**で狙い撃ち可能。相手が煙幕を使えば1ターンタグ通知を遮断できる。
> 外れた場合は情報ゼロだが、「外れた」という事実で相手の大まかな位置は推定できる。

### 9-5. 攻撃系カード（8枚）

| カード名 | ID | 燃料 | 枚数 | 効果 | 副作用 |
|---|---|---|---|---|---|
| 魚雷 | torpedo | 3 | 4 | 直線6マス攻撃、命中でHP-1 | 発射した行か列が全員に漏れる |
| 誘導魚雷 | guided_torpedo | 4 | 2 | 直線3マス後に最大2回方向転換可能、HP-1。**チャフで無効化**される | 発射時に大まかな位置が漏れる |
| 散弾 | scatter | 3 | 2 | 半径2マス全方向、HP-1。**自艦の正確な位置が全員に公開**（爆発演出・全員に見える） | 自艦位置完全公開 |

#### 攻撃3択の使い分け

| 状況 | 最適な攻撃 | 理由 |
|---|---|---|
| 追跡タグで位置確定 | **魚雷** | 確実に当てられる。射程6マスで遠距離から狙い打ち |
| だいたいの位置はわかる | **誘導魚雷** | 追尾機能で1〜2マスのズレを補正できる |
| 位置不明・とにかく当てたい | **散弾** | 半径2マスをカバー。ただし自艦位置が全員にバレる |
| 相手がチャフを持っていそう | **魚雷 or 散弾** | チャフは誘導魚雷のカウンター。直線・広域には無効 |

> **「奇跡の散弾」**: 散弾命中時は爆発エフェクトと位置情報が全プレイヤーの画面に表示される。
> ドッグファイト中（距離3以内）の至近距離では、逃げ場がなく命中率が高い。

### 9-7. ドッグファイトモード（v4.2）

```
発動条件: チェビシェフ距離 3マス以内（v4.1は距離1→v4.2で拡大）
  例: E5にいる艦とH7にいる艦 → max(|5-8|, |5-7|) = 3 → DF開始!

★ ターン1のみ例外: ゲーム開始時は海域全体に「開戦前煙幕」が張られており、
  ターン1はDF判定が発生しない。全員が安全に初期移動できる。
  ターン2以降は通常通りDF距離3ルールが適用される。

DF中の効果:
  - 両者に互いの正確な位置・向きが公開される
  - 魚雷の射程 -1（5マスに）
  - 移動コスト +1（固定コマンド前進が3燃料）
  - 第三者には「付近でDFが発生している」アイコンのみ表示

DF解除条件:
  チェビシェフ距離が 4マス以上に離れるか、どちらかが撃沈された時
```

> **v4.2での効果**: 距離3に拡大したことで「中距離接近」でもDF突入。
> ターン1の例外ルールにより初動配置の自由度を確保しつつ、ターン2から本格的な読み合いが始まる。
> 第三者は「付近のDFに漁夫の利」を狙える。

### 9-8. デッキモード別バリエーション

#### モード A：スタンダード（初心者向け）
- 標準35枚デッキ、固定コマンド常時使用可（前進2 / 旋回2）
- 収縮: ターン4から、ドッグファイト: 距離3以内（ターン1除外）
- **勝利条件**: 先に**3キル**した艦長が勝利。撃沈された艦は次ターン開始時にランダム位置で復活（総燃料50で再スタート）

#### モード B：情報戦（上級者）
- 索敵系2倍（14枚）、追跡弾を4枚に増量
- DF距離を5マスに拡大（広域ドッグファイト）
- **勝利条件**: 10ターン経過後の最多キル数で勝利（同数は総燃料残量で判定）

#### モード C：乱戦（パーティ向け）
- 攻撃系2倍（16枚）、ターン燃料上限15
- HP3スタート、収縮をターン2から
- **勝利条件**: 先に**5キル**した艦長が勝利。撃沈された艦は即次ターン復活（総燃料40）

### 9-9. 復活ルールと勝利条件（推奨設計）

```
【撃沈後の処理】
  撃沈されたターン: その場で退場。そのターンの残りは観戦のみ
  次ターン開始時: ランダム外縁マスに復活
  復活時の総燃料: min(撃沈時総燃料 + 20, 60) ← ペナルティあり
  HP: 4で復活

【復活ターンの手札（カード多め）】
  通常ターン: 2カテゴリを選んで各1枚（計2枚）
  復活ターン: 3カテゴリを選んで各1枚（計3枚）
  → 「追いつける気がする」選択肢の広さ。ただし位置はランダム外縁なので不利は残る

【勝利条件（スタンダード）】
  先に3キルした艦長が勝利
  → 「後がない状況」「逃げ切り vs 追撃」の判断が生まれる
  → 暇になる時間が最大1ターン（即復活）

【勝利条件（タイムアタック）】
  ターン10終了時に最多キル数で勝利
  同数の場合: 残HP合計 → 残総燃料の順で判定
  → 最後まで全員が戦い続けるモチベーションが保てる
```

> **設計意図**: バトロワ式「最後の1艦」は撃沈後の観戦時間が長くなりがち。
> 「キル数競争＋即復活」にすることで**全員が最後まで楽しく動ける**。
> 復活ターンはカード1枚多め（有利だが過剰ではない）＋燃料ペナルティで「死ぬリスク」も残す。

## 参考：ターン燃料10の行動例（v4.2）

固定コマンドのコスト（前進:2 / 旋回:2）での代表的な行動パターン。

| コンセプト | カードの組み合わせ | 合計燃料 |
|---|---|---|
| **旋回→直進→索敵** | 旋回(2) + 前進(2)×3 + パッシブソナー(1) | 9 |
| **旋回→直進→追跡** | 旋回(2) + 前進(2)×2 + 追跡弾(2) | 8 |
| **直進ダッシュ** | 前進(2)×5 | 10 |
| **ターボ＋索敵** | ターボ移動(2) + 旋回(2) + パッシブソナー(1) + 前進(2)×2 | 9 |
| **精密攻撃セット** | アクティブソナー(3) + 魚雷(3) + 前進(2)×2 | 10 |
| **誘導魚雷先手** | 誘導魚雷(4) + 旋回(2) + 前進(2)×2 | 10 |
| **散弾＆即逃走** | 散弾(3) + ターボ移動(2) + 旋回(2) | 7（残り3で前進or待機） |
| **チャフ→全力前進** | チャフ展開(1) + 前進(2)×4 + 旋回(2) | 11 → 前進3回に: 9 |
| **修理＋逃走** | 緊急修理(2) + 緊急離脱(4) + 前進(2) | 8 |
| **高出力ビッグムーブ（14T）** | 高出力推進(2) + ターボ移動(2=6マス!) + 前進(2=2マス!)×3 | 10/14 |
