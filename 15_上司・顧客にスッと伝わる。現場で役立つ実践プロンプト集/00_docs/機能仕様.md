# プロンプト集 VS Code Extension - 機能仕様書

## 概要
上司・顧客にスッと伝わる。現場で役立つ実践プロンプト集を VS Code 上で快適に使えるようにする Extension。
**「継続が楽しくなる」** をコンセプトに、使用履歴の可視化とモチベーション維持機能を提供。

---

## 1. コア機能

### 1.1 サイドバーパネル
- **プロンプト一覧表示**
  - カテゴリ別に整理（フォルダ構造を反映）
  - 検索・フィルタリング機能
  - お気に入り登録
  
- **プレビュー機能**
  - プロンプト内容のプレビュー
  - サンプル出力の表示
  - 使用回数バッジ表示

### 1.2 プロンプト操作
- **ワンクリック操作**
  - クリップボードにコピー
  - エディタに直接挿入
  - 新規ファイルとして開く
  
- **サンプルファイル連携**
  - プロンプトと一緒にサンプルも開く
  - Before/After の比較表示

### 1.3 テンプレート管理
- **新規プロンプト作成**
  - 標準フォルダ構成の自動生成
  - テンプレートファイルの自動配置

## 2. 継続を楽しくする機能 🎯

### 2.1 使用履歴の記録
**履歴フォルダ構造**
```
.mentor-ai/
├── history/
│   ├── 2026/
│   │   ├── 02/
│   │   │   ├── 15/
│   │   │   │   ├── session-001.md
│   │   │   │   ├── session-002.md
│   │   │   │   └── daily-summary.json
│   │   │   └── monthly-summary.json
│   │   └── yearly-summary.json
│   └── stats.json
```

**記録内容**
- 使用したプロンプト名
- 使用日時
- 使用回数
- セッション時間
- 生成したファイルへのリンク

### 2.2 可視化・統計機能

#### **ダッシュボード表示**
- 📊 **使用統計**
  - 今日/今週/今月の使用回数
  - よく使うプロンプト TOP 5
  - 連続使用日数（ストリーク）
  
- 📈 **成長の可視化**
  - プロンプト使用のヒートマップ（GitHub風）
  - カテゴリ別使用分布
  - 月間アクティビティグラフ

#### **達成バッジシステム**
- 🏆 **マイルストーン**
  - 初回使用: "First Step"
  - 10回使用: "Getting Started"
  - 50回使用: "Regular User"
  - 100回使用: "Power User"
  - 7日連続: "Week Warrior"
  - 30日連続: "Monthly Master"

- 🎖️ **プロンプト別バッジ**
  - 各プロンプトを5回使用で専用バッジ
  - 全プロンプト制覇で "Complete Master"

### 2.3 振り返り機能

#### **セッション記録**
各使用時に自動記録：
```markdown
# セッション記録 - 2026-02-15 09:35

## 使用プロンプト
- 報告書作成プロンプト

## 生成ファイル
- [報告書_v1.md](./報告書_v1.md)
- [報告書_v2.md](./報告書_v2.md)

## メモ
（オプション：ユーザーが追記可能）
```

#### **デイリーサマリー**
1日の終わりに自動生成：
```json
{
  "date": "2026-02-15",
  "totalSessions": 3,
  "promptsUsed": ["報告書作成", "返信作成"],
  "filesCreated": 5,
  "totalTime": "45min",
  "streak": 7
}
```

#### **振り返りビュー**
- カレンダービューで過去の活動を確認
- 各日をクリックでセッション詳細表示
- 週次・月次レポートの自動生成

### 2.4 モチベーション維持機能

#### **通知・リマインダー**
- 連続記録が途切れそうな時の優しいリマインド
- 新しいマイルストーン達成時の祝福メッセージ
- 週次レポートの自動表示

#### **目標設定**
- 週間目標の設定（例：週5回使用）
- 進捗バーの表示
- 目標達成時の視覚的フィードバック

#### **共有機能**（オプション）
- 月次レポートの画像エクスポート
- 統計データのMarkdown出力
- SNS共有用の画像生成

## 4. 技術仕様

### 4.1 使用技術
- **Extension API**: VS Code Extension API
- **UI**: Webview (HTML/CSS/JavaScript)
- **データ保存**: ワークスペース設定 + ローカルファイル
- **グラフ描画**: Chart.js または D3.js

### 4.2 データ構造

#### **stats.json**
```json
{
  "totalUsage": 45,
  "currentStreak": 7,
  "longestStreak": 10,
  "badges": ["first_step", "week_warrior", "regular_user"],
  "promptStats": {
    "報告書作成": {
      "count": 12,
      "firstUsed": "2026-02-08T10:30:00Z",
      "lastUsed": "2026-02-15T09:35:00Z"
    }
  },
  "dailyActivity": {
    "2026-02-15": 3,
    "2026-02-14": 5
  }
}
```

#### **session-001.md**
```markdown
---
date: 2026-02-15T09:35:00Z
prompt: 報告書作成プロンプト
duration: 15min
---

# セッション記録

## 使用プロンプト
報告書作成プロンプト

## 生成ファイル
- [報告書_draft.md](link)

## 振り返りメモ
（ユーザーが後で追記可能）
```

### 4.3 設定項目
```json
{
  "promptCollection.historyPath": ".prompt-history",
  "promptCollection.enableNotifications": true,
  "promptCollection.dailyGoal": 3,
  "promptCollection.weeklyGoal": 15,
  "promptCollection.showBadges": true,
  "promptCollection.autoSaveSession": true
}
```

## 6. ユーザーストーリー

### ストーリー1: 初めての使用
```
1. Extension をインストール
2. サイドバーにプロンプト一覧が表示される
3. 「報告書作成」をクリック
4. プロンプトがエディタに挿入される
5. 🎉 "First Step" バッジ獲得！
```

### ストーリー2: 継続使用
```
1. 毎日プロンプトを使用
2. ダッシュボードで連続日数を確認
3. 7日目に "Week Warrior" バッジ獲得
4. ヒートマップが緑色に染まっていく
5. モチベーション維持！
```

### ストーリー3: 振り返り
```
1. 月末にダッシュボードを開く
2. 今月の使用統計を確認
3. よく使ったプロンプトを分析
4. 成長を実感
5. 来月の目標を設定
```

## まとめ

この Extension は単なるプロンプト管理ツールではなく、
**「継続的な成長を楽しく可視化するパートナー」** です。

使用履歴の記録と可視化により：
- 📈 成長を実感できる
- 🎯 モチベーションを維持できる
- 🏆 達成感を得られる
- 📊 振り返りができる

プロンプトを使うたびに、自分の成長が目に見える形で積み重なっていく。
それが、継続を楽しくする鍵です。
