# Mentor AI - プロンプト集

上司・顧客にスッと伝わる。現場で役立つ実践プロンプト集を VS Code で快適に使える拡張機能です。

## ✨ 特徴

### 📚 6つの実践プロンプト
- **報告書作成プロンプト** - AIに「質問」してもらい、自分で気づく
- **返信作成プロンプト** - 相手の気持ちを読み解き、最初に応える
- **方針提案プロンプト** - 遅延報告に「線引き」をつけて提案
- **落とし所提案プロンプト** - A案とB案を提示し、落とし所を示す
- **結論ファースト調査プロンプト** - 結論を最初に。実況中継禁止
- **ヒント出し回答プロンプト** - 手順書ではなく設定ファイルを渡す

### 🎯 継続を楽しくする機能
- **📊 使用履歴の可視化** - ヒートマップ（GitHub風）で成長を実感
- **🏆 バッジシステム** - マイルストーン達成でバッジ獲得
- **🔥 ストリーク機能** - 連続使用日数を記録・表示
- **📈 統計ダッシュボード** - 使用回数、TOP 5プロンプトなど
- **📝 セッション記録** - 毎回の使用を自動記録
- **🎖️ 達成通知** - バッジ獲得時に祝福メッセージ

## 🚀 使い方

### インストール
1. VS Code のサイドバーから拡張機能を検索
2. "Mentor AI - プロンプト集" をインストール

### 初回セットアップ
1. ワークスペースを開く
2. Extension がアクティブになると、セットアップを促すメッセージが表示される
3. 「はい」をクリックすると、`.mentor-ai` フォルダが作成される
   - `templates/` - プロンプトテンプレート
   - `history/` - 使用履歴（自動生成）

### 基本操作
1. サイドバーに表示される「🎯 プロンプト集」アイコンをクリック
2. 使いたいプロンプトを選択
3. 📋 コピー、または ✏️ 挿入ボタンをクリック

### プロンプトの使用
1. プロンプトをクリップボードにコピー
2. AI チャット（ChatGPT、Claude など）に貼り付け
3. 作成した文書を貼り付けてフィードバックをもらう
4. 使用するたびに自動で履歴が記録される

### ダッシュボードを見る
- サイドバーの「ダッシュボード」セクションをクリック
- 使用統計、バッジ、ヒートマップを確認
- 継続のモチベーションアップ！

## 📊 機能詳細

### プロンプト管理
- ワンクリックでコピー・挿入
- プロンプト詳細の表示
- 使用回数バッジ（⚡×12）
- お気に入り登録

### 履歴管理
履歴は `.mentor-ai/history/` に保存されます：
```
.mentor-ai/
├── history/
│   ├── 2026/
│   │   └── 02/
│   │       └── 15/
│   │           ├── session-001.md
│   │           ├── session-002.md
│   │           └── daily-summary.json
│   └── stats.json
```

### 統計表示
- **今日の使用回数** - 本日の活動量
- **連続使用日数** - ストリーク表示（🔥）
- **累計使用回数** - 総合計
- **ヒートマップ** - 過去90日の活動
- **TOP 5プロンプト** - よく使うプロンプト

### バッジシステム
- 🎉 **First Step** - 初回使用
- ⚡ **Getting Started** - 10回使用
- 🌟 **Regular User** - 50回使用
- 💪 **Power User** - 100回使用
- 🔥 **Week Warrior** - 7日連続使用
- 👑 **Monthly Master** - 30日連続使用

## ⚙️ 設定

VS Code の設定から以下をカスタマイズ可能：

```json
{
  "mentorAi.historyPath": ".mentor-ai/history",
  "mentorAi.enableNotifications": true,
  "mentorAi.dailyGoal": 3,
  "mentorAi.weeklyGoal": 15,
  "mentorAi.showBadges": true,
  "mentorAi.autoSaveSession": true
}
```

- `historyPath` - 履歴保存パス
- `enableNotifications` - 通知の有効/無効
- `dailyGoal` - 1日の目標使用回数
- `weeklyGoal` - 1週間の目標使用回数
- `showBadges` - バッジ表示の有効/無効
- `autoSaveSession` - セッション自動保存

## 🎓 プロンプトの考え方

この拡張機能のプロンプトは、**「AIに答えを教えてもらう」のではなく「AIに質問してもらう」** という考え方に基づいています。

### なぜ「質問」してもらうのか？
- AIが答えを教える → あなたは考えない → 覚えない
- AIが質問する → あなたが考える → 気づく → 覚える

### 継続的な成長
- 使用履歴を記録
- 自分のパターンを把握
- 次回に活かす
- 同じミスを防ぐ

## 📝 セッション記録の活用

各使用時に自動で記録されるセッションファイルには、後からメモを追記できます：

```markdown
---
date: 2026-02-15T09:35:00Z
prompt: 報告書作成プロンプト
---

# セッション記録

## 使用プロンプト
報告書作成プロンプト

## 生成ファイル
- report_v1.md
- report_v2.md

## 振り返りメモ
今回は「いつまでに終わるか」を忘れていた。
次回は最初にチェックする。
```

### よくある質問

**Q: プロンプトを編集できますか？**
A: はい、`.mentor-ai/templates/` の各フォルダ内のファイルを編集できます。編集後は「更新」ボタンで反映してください。

**Q: 履歴データはどこに保存されますか？**
A: `.mentor-ai/history/` フォルダに保存されます（ローカル）。

**Q: チーム共有できますか？**
A: `.mentor-ai/templates/` フォルダを Git で共有できます。`.mentor-ai/history/` は `.gitignore` に追加することをおすすめします。

**Q: テンプレートを最新版に更新するには？**
A: コマンドパレット（Ctrl+Shift+P）から「Mentor AI: テンプレートを更新」を実行してください。

**Q: AI は含まれていますか？**
A: いいえ、この拡張機能はプロンプト管理と履歴記録のみです。ChatGPT や Claude などの AI サービスと併用してください。
