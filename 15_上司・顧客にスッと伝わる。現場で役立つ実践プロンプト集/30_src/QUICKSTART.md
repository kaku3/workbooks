# 🚀 クイックスタートガイド

Mentor AI プロンプト集 Extension へようこそ！
このガイドでは、5分で Extension を使い始める方法を説明します。

## 📦 インストール

### 必要なもの
- VS Code 1.85.0 以上
- Node.js 20.x 以上（開発時のみ）

### インストール手順

#### 方法1: VSIXパッケージからインストール（推奨）
```powershell
# まずパッケージをビルド
cd mentor-ai\30_src
npm install
npm run compile

# VSIXパッケージを作成（npx を使用 - グローバルインストール不要）
npx @vscode/vsce package

# VS Code で開く
# Extensions > "..." > VSIX からインストールを選択
# 生成された .vsix ファイルを選択
```

**注意:** Volta などの Node.js バージョン管理ツールを使用している場合、
`npm install -g` でエラーが出ることがあります。`npx` を使えば問題ありません。

#### 方法2: 開発モードで実行
```powershell
cd mentor-ai\30_src
npm install
npm run watch

# VS Code で mentor-ai\30_src フォルダを開く
# F5 キーを押してデバッグ起動
```

## 🎯 初回セットアップ

### 1. ワークスペースを開く
Extension はワークスペース内に `.mentor-ai` フォルダを作成します。
任意のフォルダをワークスペースとして開いてください。

### 2. セットアップダイアログ
Extension がアクティブになると、「セットアップを開始しますか？」と表示されます。
「はい」をクリックすると：
- `.mentor-ai/templates/` にプロンプトテンプレートがコピーされる
- `.mentor-ai/history/` に履歴フォルダが作成される
- `.mentor-ai/README.md` が生成される

### 3. サイドバーを開く
左側のアクティビティバーに「🎯」アイコンが表示されます。
クリックしてプロンプト集のサイドバーを開きます。

### 4. プロンプトを確認
6つのプロンプトが一覧表示されます：
- 📁 報告書作成
- 📁 返信作成
- 📁 方針提案
- 📁 落とし所提案
- 📁 結論ファースト調査
- 📁 ヒント出し回答

## 📋 基本的な使い方

### プロンプトをコピーする
1. 使いたいプロンプトをクリック
2. 右側に表示される 📋 コピーボタンをクリック
3. クリップボードにコピーされます
4. ChatGPT や Claude に貼り付けて使用

### プロンプトを挿入する
1. エディタでファイルを開く
2. プロンプトを挿入したい場所にカーソルを移動
3. サイドバーでプロンプトをクリック
4. ✏️ 挿入ボタンをクリック
5. カーソル位置にプロンプトが挿入されます

### プロンプト詳細を見る
1. プロンプトをクリック
2. 詳細ビューが開きます
3. プロンプト内容、説明、使用統計を確認

## 📊 ダッシュボードを見る

### 統計を確認
1. サイドバーの「ダッシュボード」セクションをクリック
2. 使用統計が表示されます：
   - 🔥 連続使用日数
   - 📈 今日の使用回数
   - 📊 累計使用回数
   - ヒートマップ（過去90日）
   - TOP 5 プロンプト

### バッジを集める
プロンプトを使うたびにバッジを獲得できます：
- 🎉 First Step（初回使用）
- ⚡ Getting Started（10回使用）
- 🌟 Regular User（50回使用）
- 💪 Power User（100回使用）
- 🔥 Week Warrior（7日連続）
- 👑 Monthly Master（30日連続）

## 🎓 実践例：報告書作成

### Step 1: プロンプトをコピー
```
1. サイドバーで「報告書作成」をクリック
2. 📋 コピーボタンをクリック
```

### Step 2: 報告書を作成
```
1. エディタで報告書を書く
2. 例：「プロジェクトの進捗報告.md」
```

### Step 3: AI にチェックしてもらう
```
1. ChatGPT を開く
2. コピーしたプロンプトを貼り付け
3. 報告書の内容を貼り付け
4. AI が質問形式でフィードバック
```

### Step 4: 成長を確認
```
1. ダッシュボードを開く
2. 使用回数が増えていることを確認
3. ヒートマップに色がついていることを確認
```

## ⚙️ おすすめ設定

### 通知を有効にする
```json
{
  "mentorAi.enableNotifications": true,
  "mentorAi.showBadges": true
}
```

### 目標を設定する
```json
{
  "mentorAi.dailyGoal": 3,
  "mentorAi.weeklyGoal": 15
}
```

### 自動保存を有効にする
```json
{
  "mentorAi.autoSaveSession": true
}
```

## 💡 使いこなすコツ

### 1. 習慣化する
- 毎日同じ時間にプロンプトを使う
- ストリークを意識する
- ダッシュボードで成長を確認

### 2. セッション記録を活用
```markdown
# 履歴フォルダ
.mentor-ai/history/2026/02/15/session-001.md

# 後から振り返りメモを追記
## 振り返りメモ
今回は「いつまでに」を忘れがち。
次回は最初にチェックする。
```

### 3. お気に入りを活用
- よく使うプロンプトを ⭐ お気に入りに追加
- サイドバーの上部に表示される
- クイックアクセスが可能

### 4. プロンプトをカスタマイズ
```
.mentor-ai/templates/ の各フォルダを開く
↓
概要.md、トレーナープロンプト.md、まとめプロンプト.md を編集
↓
自分の使いやすい形にカスタマイズ
↓
コマンドパレット > 「Mentor AI: プロンプト一覧を更新」で反映
```

## 🔧 トラブルシューティング

### プロンプトが表示されない
- ワークスペースが正しく開かれているか確認
- 各フォルダに `90_prompt.md` があるか確認
- 「更新」ボタンをクリック

### ダッシュボードが空
- まずプロンプトを1回使ってみる
- 自動で統計が記録される
- ダッシュボードを再読み込み

### 統計がリセットされた
- `.mentor-ai/history/stats.json` を確認
- バックアップがあれば復元
- 通常は自動保存される
