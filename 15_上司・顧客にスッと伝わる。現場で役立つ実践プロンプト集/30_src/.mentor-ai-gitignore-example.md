# .mentor-ai フォルダについて

このフォルダは、ワークスペースに作成されるプロンプトテンプレートと履歴を保存するフォルダです。

## Git 管理について

### 推奨設定

```.gitignore
# Mentor AI の履歴は個人データなので除外
.mentor-ai/history/
.mentor-ai/stats.json

# テンプレートは共有したい場合は含める（推奨）
# .mentor-ai/templates/
```

### チームで共有する場合

テンプレートをカスタマイズしてチームで共有したい場合：
1. `.mentor-ai/templates/` を Git に含める
2. `.mentor-ai/history/` は `.gitignore` に追加
3. チームメンバーは pull 後、Extension が自動で認識

### 個人利用の場合

`.mentor-ai/` フォルダ全体を `.gitignore` に追加してもOKです。
その場合、各自の環境で初回セットアップが実行されます。
