# VS Code Extension : 上司・顧客にスッと伝わる。現場で役立つ実践プロンプト集

## これは何？

**「AIに質問してもらう」** ことで、思考を鍛える実践プロンプト集です。

上司に提出したら「なんか違う」と言われる。返信を書くのに時間がかかりすぎる。提案しても相手の反応がイマイチ——そんな現場のコミュニケーション課題を、AIとの対話で改善していくためのツールです。

VS Code 上でプロンプトを簡単に利用でき、使用履歴を可視化することで **「継続が楽しくなる」** 設計になっています。

---

## なぜ今、これが必要なのか？

### 誰も育ててくれない時代が来た

かつては「先輩の背中を見て育つ」キャリアパスが存在しました。しかし今、現場には「育てる余裕」も「育て方のノウハウ」もありません。

- 上司に何度も聞き返すのは気が引ける
- 先輩に添削を頼むのは相手の時間を奪う
- でも失敗は許されない

このような八方塞がりの状況で、**自分で自分を育てる力** が求められています。

### AIは「心理的に安全な練習場」

AIなら、何度質問しても嫌な顔ひとつしません。

上司に出す前に、AIに叩いてもらう。  
顧客に送る前に、AIに添削してもらう。  
「これ、相手にどう映るか？」を、AIに予測してもらう。

本番前のリハーサル相手としてAIを使えば、誰の時間も奪わず、心理的にも削れることなく、何度でも試行錯誤できます。

### 「答え」ではなく「思考」を鍛える

重要なのは、**AIに答えを出させるのではなく、思考の甘さを指摘してもらうこと** です。

「この書き方だと、相手はどう受け取りますか？」  
「もっと伝わる言い方はありますか？」  
「この報告、上司は何を知りたがっていますか？」

こうやって、人とやり取りできる考え方を、AIとのやり取りで学んでいきます。

## メンター・メンティ両方に役立つ

### メンティ（教わる側）にとって
- 上司や先輩の時間を奪わずに、何度でも練習できる
- 自分のペースで思考を整理できる
- 使用履歴とバッジで、成長を実感できる

### メンター（教える側）にとって
- 後輩に「まず、これで自分の考えを整理してから質問してね」と渡せる
- 後輩の苦手なポイントに合わせてプロンプトをカスタマイズできる
- **あなたの観察眼 × AIの反復力 = 最強の育成システム**
- 本当に必要な時だけ、指導に集中できる

---

## 何ができるのか？

### 📚 6つの実践プロンプト

1. **報告書作成** - 事実の羅列から脱却。抽象度をコントロールする
2. **返信作成** - 相手の気持ちを読み解き、最初に応える
3. **方針提案** - 板挟みを抜け出す。方針で主導権を握る
4. **落とし所提案** - A案とB案を提示し、現実的に合意する
5. **結論ファースト調査** - 苦労話ではなく、結論で動かす
6. **ヒント出し回答** - 手取り足取りではなく、自立を促す

### 📊 継続を楽しくする機能

- **ヒートマップ表示**（GitHub風）- 週間・月間の活動を可視化
- **バッジシステム** - マイルストーン達成で「成長」を実感
- **使用履歴の自動記録** - セッションごとに記録、後から振り返り
- **ストリーク（連続日数）** - 継続のモチベーションを維持

## さっそく使ってみる

### 🎯 すぐに使いたい人向け（1分でインストール）

ビルド不要！パッケージ済みのExtensionをインストールするだけ：

1. **vsixファイルを取得**
   ```
   📁 60_release/
     └── mentor-ai-prompt-collection-0.1.0.vsix  ← これをインストール
   ```

2. **VS Codeでインストール**
   - VS Code を開く
   - 拡張機能タブ（Ctrl+Shift+X）を開く
   - 右上の「…」メニュー → 「VSIXからのインストール」
   - `60_release/mentor-ai-prompt-collection-0.1.0.vsix` を選択

3. **使い始める**
   - サイドバーに「🎯」アイコンが表示される
   - クリックしてプロンプト一覧を確認！

詳しい使い方は **[30_src/QUICKSTART.md](30_src/QUICKSTART.md)** をご覧ください。
