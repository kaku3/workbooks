| 観点 | 評価 | 詳細 |

## 9. チーム内受け渡し戦略の設計考察

**ユーザーの洞察：**
> 解除班：逮捕できそうな方に解除キットを集める  
> 爆弾魔：爆破できそうな方に爆弾パーツを集める  
> ＝チーム内での受け渡しがカギ・間違って渡すと相手が有利に

この読みは正しい。「チームで最もターゲットに近い1人にアイテムを集中させる」という戦略は、以下のゲームメカニクスと直接噛み合う：

| 動機 | ゲームへの影響 |
|---|---|
| **アタッカー1人に集中させた方が速い** | 2人に分散するとどちらも勝利条件に届かない。「誰が出る？」の事前合意が重要 |
| **ぐるぐる回る意味が生まれる** | 工場・ジャンク屋を繰り返し踏んでアイテムを稼ぎ、最終的にアタッカーへ渡す役割が成立する |
| **受け渡し失敗のリスクが生まれる** | チームメイトだと思って渡した相手が実は敵 → 敵の勝利条件を自分で揃えてしまう最悪の失敗パターン |
| **探知機・警察本部との連携** | 探知機で「あいつが怪しい」と気づいたメンバーが、行動でチームに伝えながら追い込む |

**「誰がアタッカーか」の見えにくさが情報戦になる：**

- 爆弾魔チームでは1人がアタッカー（パーツ3種揃え担当）、もう1人はサポート・撹乱役という分業が自然に発生する
- 解除班からすると「どちらが本命のアタッカーか」を見極めるのも戦略になる
- 本命アタッカーに見せかけた囮を立てて、真のアタッカーが動く陽動戦略が生まれる

## Q1. ちゃんと決着がつくか

### ✅ 主要な詰みパターンは解消

- **タワー永続封鎖** → 通行止め不可で解消済み
- **アイテム閉じ込め** → 下記の循環で膠着は稀

入手したアイテムが特定プレイヤーの手に固定されると膠着しうる。

> 循環経路：アイテムパイル → 手持ち → 取引/強奪/横流し/ポイ捨て → 手持ち or 捨て札（闇市で再取得可）

- アイテムは捨て札に落ちないと再循環しない
- 4人いれば取引・強奪で動かし続けられるため、完全膠着は稀

**決着がつかない可能性：低**（ただしターン上限の実装を推奨）

> 📌 **未実装タスク：40ターン上限** → 到達時点でパーツを最も多く持つ爆弾魔チームが勝利

## Q2. 爆弾魔 vs 解除班、どちらかが一方的に有利か

### ✅ ほぼ均衡

現行構成での主なバランス要素：

| 要素 | 効果 |
|---|---|
| タワー・スタート封鎖不可 | 爆弾魔が常に起爆ルートを持つことを保証 |
| アイテムパイル分離 | 初期手札の運差ゼロ。対称スタート |
| 警察本部（マス14）マス効果 | 確定で役職確認。踏まなければバレない |
| ダミー4枚 | アイテム所持だけでは役職断定不可 |
| キットX/Y/Z 3種揃え | 解除班の勝利条件として適切な難度 |

#### 残る課題

| 観点 | 内容 |
|---|---|
| **工場サーチ妨害** | 解除班が爆弾魔の欲しいパーツを意図的に取得して塩漬けできる。キット収集と二択になるトレードオフはあるが強力。 |
| **警察本部（役職一発バレ）** | マサ14を踏むと確定情報で役職が判明。自分が踏む必要なため常に倫えるわけではないが、踏まれた場合のリスクが爆弾魔にとって大きい。 |
| **チーム連携の非対称** | 解除班は「キットを集め始めた2人」の行動で担当制に気づきやすいが、爆弾魔は何も発信できない。 |

## Q3. 駆け引きは発生するか

### ✅ 情報戦の密度が高い

| シーン | 評価 | 詳細 |
|---|---|---|
| **パーツ工場サーチ** | ★★★ | 何を取ったか周囲には分からない。爆弾魔がキット取得でアリバイ偽装も可 |

## シナリオ A：警察本部後の心理戦（役職確定情報から動く）

```
Turn 3: 解除班P1が警察本部（マス14）に到達 → P3が「爆弾魔」と確定（P1のみ知る）
Turn 4: P1はP3を牽制するが、言動でバレかねないため行動観察のみで絞り込む。
        P3は「自分は解除班だ」とアピールするため、工場でキットXを意図的に取る（アリバイ作り）
Turn 5: 爆弾魔P4が「P3は怪しくないんじゃないか？」と煙幕を張る
Turn 6: P1はP3とP4のどちらに集中すべきか悩む（確定情報はP3だが、P4が動いている）
Turn 7: P3がジャンク屋でパーツBを取得。P1が追跡して強奪
Turn 9: 解除班P2がキットX/Y/Z 全3種揃い → P3を同マスで捕捉
Turn 11: P3はパーツ2種しか揃えられず確保宣言 → 解除班勝利
```

**評価：** 警察本部は確定情報だが「自分だけ知る」ため、チームへの伝え方が鍵になる。  
P3のアリバイ偽装・P4の煙幕が機能し、解除班は「確定情報をいつ・どう活かすか」の判断ゲームになる。

```
Turn 2: 爆弾魔P1がパーツ工場でパーツAをサーチ取得

        キットはP3が1枚・P4が1枚しか持っていない
```

**評価：** 探知機が引けなかった場合の早期決着は依然として発生しうる。  
工場サーチを2回使えば3種揃えられるため、解除班がキットを集めるペースとの競争になる。  
工場マス（マス4）の到達しやすさとコース設計のバランスが重要。

## 12. 確保失敗ルール シミュレーション

> **変更案 J：** 確保宣言時にキットを消費する。対象が解除班（味方）だった場合はキットをアイテムパイルに戻し、再収集する必要がある。

---

### ルール変更点

| 項目 | 現ルール | 変更後 |
|---|---|---|
| 確保宣言のコスト | キット3種保持＋同マスで宣言。成功・失敗ともコストなし | **失敗時：キット3種をアイテムパイルに返却。再収集が必要（4〜7ターンのロス）** |
| 成功時（相手が爆弾魔）| 解除班勝利（キット消費なし） | 変更なし。勝利確定なのでキット返却は関係なし |
| 失敗コスト | **なし（現状リスクゼロ）** | キット3枚がパイルに返却。再収集が必要 |

**意図：** 確証なしの確保宣言を抑制する。探知機情報やアイテム所持観察の重要度を上げる。

---

### シナリオ J-1：探知機情報不足での誤逮捕

```
Turn 7: 解除班P3がキット3種揃い（X, Y, Z）
        探知機結果：P1が「やや怪しい（★★）」—— 70%程度の確信度
Turn 8: P3が「多分そうだろう」とP1と同マスで確保宣言
        → P1は解除班だった。キットX/Y/Zがアイテムパイルに返却
Turn 9: P3が工場・ジャンク屋で再収集開始
Turn 10: 爆弾魔P4（本命）がパーツABC揃い → タワーへ
Turn 12: 解除班P3/P4のキットは合計1種。確保宣言不可 → 起爆。爆弾魔勝利
```

**評価：** ★★の探知機情報を確証として動くと致命傷になる。「★★★確認まで待つ」という判断が重要になる。  
一方、待ちすぎると爆弾魔がパーツを揃え終わる — **「いつ動くか」のタイミング判断に緊張が生まれる。**

---

### シナリオ J-2：爆弾魔の「誘い込み作戦」（5人以上で有効）

5人プレイ（爆弾魔P1, P2 / 解除班P3, P4, P5）での展開例：

```
Turn 4: 爆弾魔P2（サポート役）がアイテム工場を頻繁に往復し、
        いかにも「パーツを収集している」ような動きを見せる。実際はダミーを多く保有。
Turn 5: 解除班P3が探知機 → P2が「怪しい（★★★）」（爆弾魔なので正解）
Turn 6: 解除班P3がキット3種揃い → P2を追跡。同マスで確保宣言！
        → P2は確かに爆弾魔だが——パーツは1枚しか持っていない（本命はP1）
        ——確保成功、解除班勝利！……ではなく
```

> ⚠️ **現行ルールでは**「爆弾魔の確保成功」は即解除班勝利。P2がパーツをいくつ持っていても無関係。

→ つまり「爆弾魔をわざと捕まえさせる」作戦は、捕まった側が爆弾魔ならそのまま解除班の勝ちになる。

**有効な誘い込みは「解除班と見分けがつかない行動をして、相手の確保宣言を解除班に向けさせる」パターン：**

```
Turn 4: 解除班P4が工場でキット収集中。動きがパーツ収集に見えなくもない
        爆弾魔P2が「P4が爆弾魔っぽい」と情報をチームに流す（ソーシャルブラフ）
Turn 6: 解除班P3がP4を疑い始め、キット3種揃い次第P4に向けて確保宣言
        → P4は解除班。キット3種消費
Turn 7: 解除班P3は再収集中。爆弾魔P1がパーツABC揃い → タワーへ
Turn 9: 起爆。解除班勝利が遅れて爆弾魔勝利
```

**評価：** ✅ 爆弾魔がソーシャルブラフで「解除班同士の疑い」を生み出す戦略が有効になる。特に5〜6人プレイで全員の正体が把握しにくい環境で機能する。

### シナリオ J-3：確保失敗ルール ＋ 3人プレイの相性

3人プレイ（爆弾魔P1 / 解除班P2, P3）：

```

        しかし P3 は「P1が怪しい確証が★★止まり」なので慎重に様子を見る

```

→ 3人プレイでは1人が爆弾魔なので、確保ミスの深刻さが4人以上よりやや低い（対象が2択）。

### 確保失敗ルール バランス評価

| 観点 | 効果 |
|---|---|
| 探知機情報の価値 | **↑↑ 大幅上昇** — 確保前に★★★の確証がないと動けなくなる |

| 解除班の慎重度 | **↑↑** — 「今動くべきか」の判断に緊張が加わる |
| ゲームの長さ | ↑ — 確保ミスのリカバリーで延長しやすい（ターン上限と組み合わせを推奨） |
| 解除班の難易度 | ↑ やや難化（確証なし確保が致命打になる） |

#### 懸念と緩和策

| 懸念 | 緩和策 |
|---|---|
| 誤逮捕でキット全消費 → 再収集に5〜7ターン → タイムロスが大きすぎる | 「消費は1枚のみ」や「パイルではなく捨て札に落とす（闇市で再取得可）」に緩和も検討 |

| 3人プレイでの影響が軽微（どうせ2択） | **確定：3人プレイでは「確保失敗 = 1枚消費」に緩和** |
| 5〜6人で誤逮捕リスクが高い（候補多数） | **確定：5〜6人プレイでは「確保失敗 = 1枚消費」に緩和** |

> 💡 **確定ルール：** キット消費は「アイテムパイルへの返却（再入手可）」に統一。完全廃棄にすると解除班が詰みやすくなりすぎる。返却でも 5〜7 ターンのロスは維持されるため、緊張感のペナルティは十分機能する。

|---|---|---|

## 16. 変更案 U：警察本部弱体化 シミュレーション

> **解決したい課題**  
> **自陣営1人特定 → 全員特定問題** — 現行の警察本部（マス14）は指定相手の陣営を**100%確定**で知れるため、自陣営の1人が判明した時点で消去法により全員の陣営が確定してしまう。特に4人プレイでは「1人確認 = 残り全員確定」となり、役職推理の要素が一瞬で消滅する。

### シナリオ U-4：全員特定問題の解消（4人プレイ）

4人プレイ（爆弾魔P1,P2 / 解除班P3,P4）

```
【現行ルール（変更前）の問題点】
Turn 5: P3がP1を捜査 → 爆弾魔と確定
        → 自動的にP2,P4の両方が「自分と同じ解除班」か「爆弾魔」かが判明
        → P3が解除班なので、P2が爆弾魔ならP4は解除班、P2が解除班ならP4が爆弾魔
        → 4人の場合、爆弾魔2:解除班2 なので P1=爆弾魔 → P3=解除班（自分）
           → 残りP2,P4のうち1人が爆弾魔、1人が解除班
        → 1回の捜査では全員確定しないが、2回目の捜査で完全確定

Turn 7: P3がP2も捜査 → 解除班と確定
        → P4が爆弾魔と自動判明 → 全員の陣営が確定。推理ゲーム終了

【変更案Uの場合】
Turn 5: P3がP1を捜査 → P1の陣営カード1 / ダミー1 → ❌ 捜査失敗
        → P1の陣営は不明のまま。推理の余地が残る

Turn 8: P3がP2を捜査 → P2の陣営カード0 / ダミー0 → ❌ 捜査失敗（アイテムなし）
        → 2回捜査しても情報が得られない可能性がある

Turn 11: P3がP1を再捜査 → P1の陣営カード2 / ダミー1 → ✅ 特定成功
         → P1は爆弾魔と判明。ただしP2は未確認
         → P2のアイテム保有状況が不明な限り、P2を捜査しても成功するか分からない

Turn 13: P3がP2を再捜査 → P2の陣営カード1 / ダミー2 → ❌ 捜査失敗
         → P2はダミーで身を守っている。消去法では「P2が爆弾魔の可能性がある」程度の推測しかできない
```

**評価：** ✅ **「1〜2回の捜査で全員確定」という問題が解消される。** 捜査の成否がアイテム保有に依存するため、複数回捜査しても失敗する可能性があり、推理ゲームとしての寿命が延びる。消去法による自動確定も「捜査失敗」の存在で不確実になる。

### 懸念と対策

| 懸念 | 内容 | 対策案 |
|---|---|---|
| 警察本部が弱すぎて使われなくなる | 序盤は高確率で失敗するため、マス14を踏むインセンティブが低下 | 探知機（変更案L）との組み合わせで補完。探知機はヒント式なので成否に関わらず部分情報が得られる |
| ダミー集めが最適解になりすぎる | ダミーを集めること自体が目的化し、本来の勝利条件（パーツ/キット収集）が遅延 | ダミーで手札を埋めると行動カード・移動カードが減り機動力が低下。自然なペナルティが存在 |
| 情報取得手段が不足する | 警察本部の確実性が下がることで、陣営情報がほぼ得られなくなる懸念 | 探偵事務所（手札1枚確認）・尋問・公開捜査・密談がヒント源として機能。探知機（変更案L）で補完 |
| 捜査失敗の情報価値 | 「捜査失敗」自体が情報になる（相手がダミーを持っている/アイテムが少ない/陣営カードが少ない） | 失敗からの推理要素としてポジティブに作用。「失敗 = 無価値」ではなく「失敗 = 一定の推理材料」 |
| 探知機（変更案L）との整合性 | 探知機はパーツ÷(3+キット+ダミー×2)のヒント式。警察本部と探知機の役割分担は明確か | 探知機 = 確率ベースのヒント、警察本部 = 条件付き確定。**不確実なヒント vs 条件付き確定**で棲み分け成立 |

### T改 + U の組み合わせ効果

変更案T（スタート地点ランダム4枚表示）と変更案U（警察本部弱体化）を同時適用した場合の相互作用：
