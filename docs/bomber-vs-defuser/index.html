    🎮 遊び方

    1 毎ターン、山札から カードを 1枚ドロー
    2 手札から 1枚プレイ
    3 移動カード：数値分コマを進めてロケーション効果発動
    4 アクションカード：他プレイヤーの手札に干渉
    5 アイテムカード（パーツ・キット）は手に持つだけで効果発動 → 山札が尽きたら捨て札から再利用
    ⚠️ 誰が爆弾魔かは秘密！アイテムカードを集めるプレイヤーを監視せよ。詳細ルールはゲーム画面の 📋 ルール ボタンでいつでも確認できます。
    📖 カード・ロケーション詳細を見る

    ⚔️ アクションカード

    🤝取引指名プレイヤーと手札を1枚交換
    👊強奪相手の手札からランダムに1枚奔う
    👥横流し全員が手札を1枚ずつ次のプレイヤーへ渡す
    🗑️ポイ捨て相手の手札をランダムに1枚捨て
    🔍尋問相手の手札を1枚こっそり確認（本人のみ）
    📰公開捜査相手の手札を1枚全員に公開
    🤫密談相手と手札を1枚見せ合う
    ⏸️足止め相手の次のターンをスキップ
    🚧通行止め指定ロケーションを1ターン封鎖
    📡探知機爆弾を探知する
    💨ダッシュ手持ちの移動カードを選んで移動量+2で移動
    🌫️煙幕相手の手札から移動カード1枚をランダムに破棄
    🧤手袋強奪・ポイ捨て・煙幕と同時使用で効果+1（消費型）

    🗺️ ロケーション効果

    🏁スタート手札2枚捨て→相手指定→ランダム4枚から1枚奪取
    🔩ジャンク屋山札から1枚ドロー
    🏭パーツ工場アイテムパイルから最大5枚の中から1枚選ぶ
    🕵️探偵事務所相手の手札を1枚こっそり確認
    🌃闇市捨て札から最大6枚の中から1枚引き取る
    🌃裏路地捨て札から最大3枚の中から1枚引き取る
    🗼タワー山札をサーチして1枚入手 / 爆弾魔がパーツ3種揃いで勝利
    🚔警察本部陣営カード＞ダミーなら陣営特定、それ以外は捜査失敗
    🍺酒場ランダムな相手と手札1枚強制交換
    🚔交番相手の手札を1枚没収
    📺放送局手札を1枚全員公開 → 1枚ドロー
    🎰カジノダイス偶数→2枚ドロー / 奇数→1枚没収
    🚦交差点全員の手札が1枚ずつ次のプレイヤーへ
    🏥病院手札全捨て → 5枚引き直し
    🚧工事現場1回休み
