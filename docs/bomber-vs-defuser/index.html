    🃏 カード種別

    🔢 移動カード数値分コマを進める → ロケーション効果発動
    ⚡ アクションカード取引・強奪・尋問・スキャンなど
    📦 アイテムカード爆弾パーツ / 解除キット / ダミー（手に持つだけ）

    ⚔️ 主なアクションカード

    🤝 取引指名プレイヤーと手札を 1枚交換
    👊 強奪相手からランダムに 1枚奔い、不要カードを押し付ける
    👥 横流し全員が手札を 1枚ずつ左鈲に渡す
    🗑️ ポイ捗て相手の手札をランダムに 1枚捨てさせる
    🔍 尋問相手の手札を 1枚こっそり確認（本人のみ）
    📡 スキャン相手の陣営（爆弾魔/解除班）を確認（本人のみ）
    📰 公開捰査相手の手札を 1枚全員に公開
    🤛 密談相手と手札を 1枚見せ合う（味方確認）
    ⏸️ 足止め相手の次のターンをスキップ
    🚧 通行止め指定ロケーションを 1ターン封鎖

    🗺️ ロケーション効果

    🔩 ジャンク屋山札から 1枚ドロー（引いたカードを本人に通知）
    🏭 パーツ工場アイテムカードを優先ドロー
    🕵️ 探偵事務所相手の手札を 1枚こっそり確認（本人のみ）
    🗷️ 闇市捨て札から好きな 1枚を引き取る
    🗇 タワー山札をサーチして 1枚入手 / 爆弾魔がパーツ揃いで起爆
    🚔 警察本部相手の役職をこっそり確認（本人のみ）
    🎺 酒場ランダムな相手と手札をランダムに 1枚強制交換
    🚚 交番相手の手札を 1枚捘収
    📺 放送局手札を 1枚全員公開 → 1枚ドロー
    🎰 カジノダイス偶数→ 2枚ドロー / 奇数→ 1枚捕収
    🚦 スクランブル全員の手札が 1枚ずつ左鈲へ
    🏥 病院手札全捨て → 5枚引き直し
