Essential

AI 時代のコードレビュー力試験 — Essential

AI が一瞬で作った問題だらけの EC サイトのコードを読み、設計上の問題を発見・説明・改善提案する。バックエンド・フロントエンドを問わず、エンジニアとして生存するための基礎力を問う 9 問。

問数
9問（初・中・上級 各 3 問）
推奨対象
実務経験 1〜3 年
テーマ
ファイル設計・バグトレース・セキュリティ・リファクタリング
詳細を見る GitHub

Tutorial

設計ドキュメント編 — チュートリアル

何も知らないプロジェクトにアサインされたとき、どうやって全体像を把握するか。サイトマップ・機能仕様書・ER図・画面仕様書・シーケンス図をAIで作りながら、「ドキュメントを全员で共有する意味」を体験する全 5 ステップのチュートリアル。Essential の引数にもなります。

ステップ数
5ステップ（T-01〜T-05）
推奨対象
初めてプロジェクトに入る方・何から始めるか分からない方
テーマ
サイトマップ・ER図・画面仕様書・シーケンス図・PlantUML

UI 編・サイトフロー編

更新予定

UI 編と サイトフロー編はセットで公開予定。EC サイトの UI 改修・ページ遷移・状態管理を題材に、フロントエンドからフルスタックまでの設計力を問う。

UI 編（予定）

コンポーネント設計・CSS 設計・レスポンシブ・アクセシビリティ・フォームバリデーションを扱う予定。

推奨対象：フロントエンド志向のエンジニア

サイトフロー編（予定）

状態管理・ルーティング設計・API 設計・UX フローを扱う予定。

推奨対象：中級以上・フルスタック志向のエンジニア
