Essential

AI 時代のコードレビュー力試験 — Essential

仕様書なしで育った EC サイトのコードを読み、設計上の問題を発見・説明・改善提案する。バックエンド・フロントエンドを問わず、エンジニアとして生存するための基礎力を問う 9 問。

問数
9問（初・中・上級 各 3 問）
推奨対象
実務経験 1〜3 年
テーマ
ファイル設計・バグトレース・セキュリティ・リファクタリング
詳細を見る GitHub

UI 編

準備中

EC サイトの UI 改修依頼を題材に、フロントエンドの設計力を問う。コンポーネント設計・CSS 設計・レスポンシブ・アクセシビリティ・フォームバリデーションを扱う予定。

推奨対象

フロントエンド志向のエンジニア

テーマ

コンポーネント設計・レスポンシブ・アクセシビリティ

前提

Essential を受けておくとコードベースの理解が早い

サイトフロー編

準備中

検索機能・カテゴリ機能・ページ遷移の追加依頼を題材に、より広いコンテキストでの設計力を問う。状態管理・ルーティング設計・API 設計・UX フローを扱う予定。

推奨対象

中級以上・フルスタック志向のエンジニア

テーマ

状態管理・ルーティング・API 設計・UX フロー

前提

Essential / UI 編の学習を推奨
